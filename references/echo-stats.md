# Echoes

Echo-sets consists of up to 5 individual Echoes, each providing stat bonuses and have a COST value attached to them. The available COST values are 1, 3 and 4. A character can equip a COST sum of up to 12.

## Echo Stats

Each Echo always rolls one variable main stat and one fixed main stat, but the type of variable stat depends on the COST tier.

### Main Stats

| **4-COST**          |                                                |
| ------------------- | ---------------------------------------------- |
| Main Stat 1         | Scaling(ATK / DEF / HP) / Crit Rate / Crit DMG |
| Main Stat 2 (fixed) | Flat ATK                                       |

> Note: Crit Rate and Crit DMG only appear as main stats on 4‑COST Echoes.

| **3-COST**          |                                                    |
| ------------------- | -------------------------------------------------- |
| Main Stat 1         | Scaling(ATK / DEF / HP) / ER / Elemental DMG Bonus |
| Main Stat 2 (fixed) | Flat ATK                                           |

> Note: Elemental DMG Bonus only appears as a main stat on 3‑COST Echoes.

| **1-COST**          |                         |
| ------------------- | ----------------------- |
| Main Stat 1         | Scaling(ATK / DEF / HP) |
| Main Stat 2 (fixed) | Flat HP                 |

> Note: 1‑COST Echoes cannot roll offensive main stats like Crit or DMG Bonus.

### Sub stats

Each Echo can roll up to 5 sub‑stats. Sub‑stats are drawn from the shared pool:

**Sub stat pool:**

> Scaling(ATK / DEF / HP), Flat ATK, Flat DEF, Flat HP, Crit Rate, Crit DMG, Skill DMG Bonus (Basic, Heavy, Res. Skill, Res. Liberation)
